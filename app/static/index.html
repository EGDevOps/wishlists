Wishlists REST API Service

GET /wishlists to see all the wishlists.

GET /wishlists/{id} retreives a wishlist with ID id.

GET /wishlists/{id}/items retreives all the items in a wishlist with ID id.

GET /wishlists/{id}/items/{item_id} retreives a specific item with ID item_id in the wishlist with ID id.

GET /wishlists/search?q=Apple&user_id=123 to query all instances of "Apple" in wishlists with user ID 123.

PUT /wishlists/{id} updates a wishlist with ID id.

PUT /wishlists/{id}/clear/items clears the item in a wishlist with ID id.

PUT /wishlists/{id}/items/{item_id} updates a item with ID item_id in the wishlist with ID id.

DELETE /wishlists/{id} removes the wishlist with ID id.

DELETE /wishlists/{id}/items/{item_id} removes the item with ID item_id in the wishlist with ID id.

POST /wishlists creates a new wishlist.

Example Data:

    {
        "name": "wishlist 1",
        "user_id": "user 1"
    }


POST /wishlists/{id}/items creates a new item in the wishlist with ID id.

Example Data:

    {
        "id": "item 1",
        "description": "test item 1"
    }
